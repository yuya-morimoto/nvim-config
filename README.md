# Neovim config manged repository

## setup

prepared

```bash
brew install nvim
```

### 1. clone

```bash
git clone git@github.com:yuya-morimoto/nvim-config.git ~/.config/nvim/
```

### 2. start neovim

```bash
nvim
```
